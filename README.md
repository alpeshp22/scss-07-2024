## SCSS learn and implementation

=> SCSS = CSS3
=> SASS = CSS

# SCSS Powers & Means

1. Operators - like +, -, / => (16px + 10px), (26px - 10px), (32 / 2)
2. Variables - $primaryColor: #F00, $fontSize: 32rem,
3. Mixin - see in file
4. Parameters - @mixinName(parameters) -> you can add multiple parameters
5. Partials - seperate files with underscore like [_var.scss], [_mixin.scss]
